# 📖 Overview

Enterprise AI Voice Assistant is an intelligent multimodal assistant built using Google AI Studio and Gemini.

It supports voice conversations, document understanding, memory, Retrieval-Augmented Generation (RAG), AI tool calling, OCR, and Vision AI inside one unified interface.

This project demonstrates modern Generative AI engineering concepts and enterprise AI application architecture.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/Veera9966148/enterprise-ai-voice-assistant.git
```

Install Dependencies

```bash
npm install
```

Create Environment File

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run Project

```bash
npm run dev
```
